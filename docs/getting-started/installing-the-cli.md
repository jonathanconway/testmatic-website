# Installing the CLI

Get started by installing Testmatic via the CLI.

#### What you'll need

- [Node.js](https://nodejs.org/en/download/) version 16.14 or above.

#### Install Testmatic

Install globally via NPM:

```bash
npm install -g testmatic
```

You can verify the installation by running:

```bash
testmatic
```
